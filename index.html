Accessor Methods

Accessor methods return a new value or representation.

.min

The .min method returns the item that qualifies as the "maximum".

# find the lowest number: numbers = [ 5, 2, 0, -3, -99 ] numbers.min #=> -99
.max

The .max method returns the item that qualifies as the "maximum".

# find the highest number: numbers = [ 5, 2, 0, -3, -99 ] numbers.max #=> 5
.minmax

The .minmax method returns a 2-element array consisting of the minimum and maximum values.

# find the lowest and highest number: numbers = [ 5, 2, 0, -3, -99 ] numbers.minmax #=> [-99, 5]
.count

The .count method counts how many elements match the given criteria.

# count all elements numbers = [ 5, 2, 0, -3, -99 ] numbers.count #=> 5 # count even numbers numbers = [ 5, 2, 0, -3, -99 ] numbers.count(&:even?) #=> 5 # count numbers divisible by 5 numbers = [ 5, 2, 0, -3, -99 ] numbers.count{|num| num % 5 == 0} #=> 5
Subset Methods

Accessor methods return a new value or representation.

.first

The .first method returns the first n elements.

# find the first number: numbers = [ 5, 2, 0, -3, -99 ] numbers.first #=> 5 # find the first n numbers: numbers = [ 5, 2, 0, -3, -99 ] numbers.first(2) #=> [5, 2]
