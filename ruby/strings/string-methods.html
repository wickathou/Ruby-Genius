Array Methods

Ruby has over 50 methods in its Enumerable Module. Official Ruby Docs

Accessor Methods

Accessor methods return a new value or representation.

.min

The .min method returns the item that qualifies as the "maximum".

# find the lowest number: numbers = [ 5, 2, 0, -3, -99 ] numbers.min #=> -99
.max

The .max method returns the item that qualifies as the "maximum".

# find the highest number: numbers = [ 5, 2, 0, -3, -99 ] numbers.max #=> 5
.minmax

The .minmax method returns a 2-element array consisting of the minimum and maximum values.

# find the lowest and highest number: numbers = [ 5, 2, 0, -3, -99 ] numbers.minmax #=> [-99, 5]
.count

The .count method counts how many elements match the given criteria.

# count all elements numbers = [ 5, 2, 0, -3, -99 ] numbers.count #=> 5 # count even numbers numbers = [ 5, 2, 0, -3, -99 ] numbers.count(&:even?) #=> 5 # count numbers divisible by 5 numbers = [ 5, 2, 0, -3, -99 ] numbers.count{|num| num % 5 == 0} #=> 5
Subset Methods

Accessor methods return a new value or representation.

.first

The .first method returns the first n elements.

# the first number: numbers = [ 5, 2, 0, -3, -99 ] numbers.first #=> 5 # the first n numbers: numbers = [ 5, 2, 0, -3, -99 ] numbers.first(2) #=> [5, 2]
.drop

The .drop returns everything except the first n elements.

# skip the first two numbers: numbers = [ 5, 2, 0, -3, -99 ] numbers.drop(2) #=> [0, -3, -99]
.last

The .drop method returns the last n elements.

# the last number: numbers = [ 5, 2, 0, -3, -99 ] numbers.last #=> [-99] # the last n numbers: numbers = [ 5, 2, 0, -3, -99 ] numbers.last(2) #=> [-3, -99]
.find

The .find method finds the first element for which the given block returns true and returns the value of that element.

# find first even number: numbers = [ 5, 2, 0, -3, -99 ] numbers.find(&:even?) #=> 2 # find first number divisible by 3: numbers = [ 5, 2, 0, -3, -99 ] numbers.find{ |num| num % 3 == 0 } #=> 0
.find_index

The .find_index method finds the first element for which the given block returns true and returns the index of that element.

# find index of first even number: numbers = [ 5, 2, 0, -3, -99 ] numbers.find_index(&:even?) #=> 1 # find index of first number divisible by 3: numbers = [ 5, 2, 0, -3, -99 ] numbers.find_index{ |num| num % 3 == 0 } #=> 2
.find_all

The .find_all / .select method return a new array of only the elements for which the given block returns true. It's the simplest way to filter a list in Ruby.

# find all even numbers: numbers = [ 5, 2, 0, -3, -99 ] numbers.find_all(&:even?) #=> [2, 0] numbers.select(&:even?) #=> [2, 0] # find all numbers divisible by 3: numbers = [ 5, 2, 0, -3, -99 ] numbers.find_all{ |num| num % 3 == 0 } #=> [0, -3, -99] numbers.select{ |num| num % 3 == 0 } #=> [0, -3, -99]
.reject

The .reject method works the opposite of the find_all / select method. It returns a new array of the elements for which the given block returns false.

# find all odd numbers: numbers = [ 5, 2, 0, -3, -99 ] numbers.reject(&:even?) #=> [5, -3, -99] # find all negative numbers: numbers = [ 5, 2, 0, -3, -99 ] numbers.reject{ |num| num >= 0 } #=> [-3, -99]
.slice

The .slice method takes an enumerable collection and two integer parameters to return a subset of the array. The first parameter specifies what index to start at and the second parements specifies the number of elements.

# slice 3 elements starting from index 0 numbers = [ 5, 2, 0, -3, -99 ] numbers.slice(0, 3) #=> [5, 2, 0] # slice 2 elements starting from index 3 numbers = [ 5, 2, 0, -3, -99 ] numbers.slice(3, 2) #=> [-3, -99] # slice 1 element starting from index 3 numbers = [ 5, 2, 0, -3, -99 ] numbers.slice(3) #=> -3
Mutator Methods

Accessor methods change the array.

.sort

The .sort method takes an enumerable collection and returns a new collection, sorted.

# sort an array of numbers numbers = [ 5, 2, 0, -3, -99 ] numbers.sort #=> [-99, -3, 0, 2, 5]
.reverse

The .reverse method takes an enumerable collection, reverses the order of the elements.

# reverse order an array of numbers numbers = [ 5, 2, 0, -3, -99 ] numbers.reverse #=> [-99, -3, 0, 2, 5]
Iterator Methods

Iterator methods are similar to loops.

.each

The .each method takes an enumerable collection and yields each element for processing.

# all numbers numbers = [ 5, 2, 0, -3, -99 ] numbers.each{ |num| print "#{num}, " } #=> 5, 2, 0, -3, -99
.each_with_index

The .each_with_index method takes an enumerable collection yields one element at a time to the given block, along with an index number.

# all numbers with index numbers = [ 5, 2, 0, -3, -99 ] numbers.each_with_index{ |num, index| puts "#{num}, #{index}" } # output # 5, 0 # 2, 1 # 0, 2 # -3, 3 # -99, 4
.reduce

The .reduce method take an enumerable collection and return an aggregate value.

# sum of all the numbers numbers = [ 5, 2, 0, -3, -99 ] numbers.reduce(:+) #=> -95 # multiply all the numbers numbers = [ 5, 2, 0, -3, -99 ] numbers.reduce{ |sum, num| sum * num } #=> 0
.map

The .map method take an enumerable collection, runs each item through the given block, and returns a new array of the results.

# all numbers multiplied by two numbers = [ 5, 2, 0, -3, -99 ] numbers.map{ |num| num * 2} #=> [10, 4, 0, -6, -198] # the even status for all numbers numbers = [ 5, 2, 0, -3, -99 ] numbers.map(&:even?) #=> [false, true, true, false, false]
Boolean Methods

Boolean methods return a true or false.

.all?

The .all? method takes an enumerable collection and tests whether the given condition is true for every item in the collection. It returns true only if every element meets the condition.

# are all numbers even? numbers = [ 5, 2, 0, -3, -99 ] numbers.all?(&:even?) #=> false # are all numbers positive? numbers = [ 5, 2, 0, -3, -99 ] numbers.all?{ |num| num >= 0 } #=> false
.any?

The .any? method takes an enumerable collection and tests whether the given condition is true for any item in the collection. It returns true even if only one element meets the condition.

# any of the numbers even? numbers = [ 5, 2, 0, -3, -99 ] numbers.all?(&:even?) #=> true # any of the numbers 15? numbers = [ 5, 2, 0, -3, -99 ] numbers.all?(15) #=> false # any of the numbers divisible by 5? numbers = [ 5, 2, 0, -3, -99 ] numbers.all?{ |num| num % 5 == 0 } #=> true
.none?

The .none? method takes an enumerable collection and tests whether the given condition is true for 0 items in the collection.

# none of the numbers are even? numbers = [ 5, 2, 0, -3, -99 ] numbers.none?(&:even?) #=> false # none of the numbers are 15? numbers = [ 5, 2, 0, -3, -99 ] numbers.none?{ |num| num == 15 } #=> true
